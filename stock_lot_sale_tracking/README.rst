=======================
Stock Lot Sale Tracking
=======================

This module allows the user the retrieve all customer deliveries from a given
 production lot. It is really useful in case of products recalls.

Installation
============

To install this module, you need to:

#. Just install it.

Configuration
=============

To configure this module, you need to:

#. Nothing special to do.

Usage
=====

To use this module, you need to:

#. Open a production Lot + Print/Deliveries Tracking
