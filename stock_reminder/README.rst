This module gives the opportunity to the users of Odoo to get notified when the
quantity of a particular product reaches a certain threshold.

**Table of contents**

.. contents::
   :local:

Configuration
=============

Navigate to Warehouse -> Configuration -> Reminders.

As a member of the User group of the stock, you can only set reminders for
yourself.
As a member of the Manager group of the stock, you can set reminders for
everybody.

Usage
=====

Just setup the Reminders that you want on
Warehouse -> Configuration -> Reminders and the module will notify the users
registered via emails when the threshold is less than a specific quantity.

Known issues / Roadmap
======================

1) Make sure that the users do not get multiple emails for the same reminder.
2) Allow the reminders to be set on complete groups or lists of users
